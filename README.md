# AI Gaming Copilot

    ## Features
    - Twitch stream analysis
    - Gameplay monitoring
    - Real-time analytics
    - Post-game analysis

    ## Setup
    ```bash
    npm install
    npm run dev
    ```

    ## Project Structure
    - `/src` - Main application code
    - `/models` - AI models and training data
    - `/utils` - Utility functions
    - `/config` - Configuration files
